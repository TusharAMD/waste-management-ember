Di Multipurpose Demo Importer plugin allows to import demo website created with Di Multipurpose WordPress theme.

== Description ==

Di Multipurpose Demo Importer plugin allows to import demo website created with Di Multipurpose theme. simply install and activate the plugin and go to: 'WordPress Admin Panel > Appearance > Install Demo' page to import any demo website. This plugin requires the [Di Multipurpose](https://dithemes.com/di-multipurpose-free-wordpress-theme/) theme to be installed.

== Installation ==

<ol>
	<li>Download plugin zip file.</li>
	<li>Open Dashboard of your WordPress website.</li>
	<li>Open: Plugins > Add New > Upload Plugin > Browse... (select downloaded zip file) > Install Now</li>
	<li>Now simply activate plugin.</li>
</ol>

== Frequently Asked Questions ==

= How to import demo website =
Go to: 'WordPress Admin Panel > Appearance > Install Demo' page to import any demo website.

= Installed the plugin but it does not work =
